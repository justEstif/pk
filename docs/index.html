Four tools. Your agent does the filing.

Connect pk to your AI tool with pk init. Drop in a Slack thread, a doc, a decision you just made — ask your agent to organize it. It writes structured notes. You search them later.

pk_search

Ranked full-text search across every note. Ask: "what did we decide about the database?"

pk_synthesize

Dump relevant context into the session. Start informed instead of re-explaining from scratch.

pk_new

Tell your agent to save a decision, question, note, or source. Structured and findable later.

pk_lint

Validate every note for completeness. Flags missing sections, stale questions, schema drift.

Get started

1

Install

npm install -g @justestif/pk
2

Init for your AI tool

pk init myproject --harness claude

Writes the MCP config so your agent discovers pk automatically. Supports: claude · claude-desktop · cursor · omp · opencode · codex

3

Feed it. It organizes. Repeat.

Paste anything in and tell your agent to save it — it writes a typed, structured note. Or go further: connect a source MCP alongside pk and your agent can pull directly from it.
